# 项目概述

本项目用于沉淀单线激光雷达相关测试脚本、协议解析工具和现场验证方案。本轮新增的 `network_test/automation` 面向 C2/H1 网络通信测试，目标是把原测试方案落地为可重复执行的命令行自动化工具。

## 当前重点

- C2/H1 网络连通、协议响应、连续取数、通断网恢复、长稳统计。
- H1/H2 历史命名统一按 H1E0-02A 协议处理。
- 第一阶段不纳入 ROS 驱动验证，不强依赖网络损伤仪自动控制。

## 用户价值

测试人员可以通过配置文件指定雷达 IP、端口、协议命令和阈值，然后使用 Pytest 或命令行 runner 执行测试，最后得到 JSON、CSV、HTML 报告，便于量产准入、交付验收和现场排障。
