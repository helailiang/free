测试流程
1. 测试准备
1.1 设备准备

准备以下设备：

单线激光雷达
Y100SC 精密转台
工控机 / 上位机
标定目标物（细杆、反光柱、小反光片）
数据采集程序
Excel 数据记录文件
1.2 雷达参数

当前单线雷达参数如下：

参数	数值
扫描角度范围	-45° ～ 225°
总扫描角度	270°
角分辨率	0.1°
点位索引范围	0 ～ 2700

对应关系如下：

index=
0.1
∘
angle+45
∘
	​


例如：

angle	index
-45°	0
0°	450
90°	1350
225°	2700

因此：

每增加 0.1°
index 增加 1

即：

1 index = 0.1°
1.3 转台旋转方向定义

定义：

转台旋转方向	index变化
正方向（+）旋转	index +1
负方向（-）旋转	index -1

即：

转台每旋转 +0.1°

目标理论上移动到：

index + 1

转台每旋转 -0.1°

目标理论上移动到：

index - 1
2. 雷达安装与目标放置
2.1 雷达安装

将单线雷达固定安装在 Y100SC 转台上，确保：

雷达只能随转台旋转
安装牢固
雷达中心尽量与转台中心同轴
2.2 标定目标放置

在指定距离放置目标物，例如：

5m
10m
20m

推荐优先使用：

10m以上距离

目标要求：

尺寸小
边缘清晰
反射稳定
连续点数量少

推荐：

细杆
小反光柱
小反光片
3. 初始点位校准
3.1 启动雷达数据采集

启动雷达点云采集程序。

每个点至少包含：

字段	说明
index	点位索引
angle	对应角度
distance	距离值
intensity	反射强度
timestamp	时间戳
3.2 搜索目标点区域

在指定 index 范围内搜索目标点。

例如：

index 1000 ~ 1020

找到目标连续点区域。

例如：

1008
1009
1010
3.3 计算目标中心点

计算目标中心点 index：

centerIndex=
2
startIndex+endIndex
	​


例如：

startIndex = 1008
endIndex = 1010

centerIndex = 1009

该点作为：

初始校准点
3.4 记录初始点位信息

记录当前点位全部信息：

字段	示例
step	0
rotateAngle	0°
targetIndex	1009
angle	55.9°
distance	10.023m
intensity	188
timestamp	xxx

同时记录：

连续点范围
起始 index
结束 index
4. 转台旋转测试
4.1 转台旋转 0.1°

控制 Y100SC 转台旋转：

+0.1°

或：

-0.1°

等待转台稳定。

4.2 查找目标新点位

根据旋转方向：

正方向旋转（+）

理论目标位置：

newIndex=oldIndex+1

例如：

1009 → 1010
负方向旋转（-）

理论目标位置：

newIndex=oldIndex−1

例如：

1009 → 1008
4.3 检查目标点是否存在

检查理论 index 附近是否存在有效目标点。

判断条件：

距离有效
强度有效
连续点存在
点位稳定
4.4 记录当前点位信息

记录：

字段	示例
step	1
rotateAngle	+0.1°
targetIndex	1010
angle	56.0°
distance	10.025m
intensity	190
timestamp	xxx

同时记录：

连续点范围
起始 index
结束 index
index 偏移量
5. 连续旋转测试
5.1 重复旋转

持续控制转台：

每次旋转 0.1°

例如：

0.1°
0.2°
0.3°
...
5°
5.2 每次旋转均执行以下步骤

每次旋转后：

（1）等待转台稳定

避免机械振动影响采样。

（2）查找理论目标 index

根据旋转方向：

正方向：
index +1
负方向：
index -1
（3）检查目标点是否存在

确认理论 index 位置是否存在有效点。

（4）记录目标点所有信息

记录：

当前 step
当前累计旋转角度
当前目标 index
angle
distance
intensity
timestamp
（5）计算 index 偏移量

Δindex=currentIndex−initIndex

（6）计算雷达测得角度

measuredAngle=Δindex×0.1
∘

（7）计算角度误差

error=measuredAngle−rotateAngle

6. 数据保存
6.1 Excel 数据保存

每次旋转后的数据均保存至 Excel。

建议字段：

step	转台角度	targetIndex	index偏移	雷达测得角度	理论角度	角度误差	distance	intensity	timestamp

例如：

step	转台角度	targetIndex	index偏移	雷达测得角度	理论角度	角度误差
0	0°	1009	0	0°	0°	0°
1	+0.1°	1010	+1	0.1°	0.1°	0°
2	+0.2°	1011	+2	0.2°	0.2°	0°
6.2 异常数据记录

若出现以下情况需单独记录：

index 未变化
index 跳变
点位丢失
连续点异常
距离突变
强度异常
7. 测试结果分析

测试完成后统计：

平均角度误差
最大误差
最小误差
标准差
丢点率
index 跳变次数

分析：

雷达角度精度
index 稳定性
长时间累计误差
正反方向一致性
连续旋转稳定性
8. 最终测试目标

最终验证：

当转台每旋转 ±0.1° 时，目标点是否稳定移动 ±1 个 index，并在长时间连续旋转过程中保持稳定一致的角度映射关系。
