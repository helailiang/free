4. 当前测试目的：

      ​	雷达转速为30HZ, 角分辨率 0.1°，通过控制Y100SC转台转0.1度，

      ​	本测试用于验证单线激光雷达的角度测量精度与角分辨率准确性。已知雷达角分辨率为 0.1°，理论上当 Y100SC 转台每旋转 0.1° 时，雷达点云中的目标点位应对应移动 1 个角度点（1 个 index）。通过控制转台精确旋转固定角度，并对比目标在旋转前后的点位变化情况，验证雷达测得的角度变化是否与转台实际旋转角度一致，从而评估雷达的角度精度、点位稳定性及重复性

5. 测试流程为：

      # 测试流程

      ------

      # Step 1：固定雷达

      将雷达固定在 Y100SC 转台上。

      确保：

      - 雷达只能随转台旋转
      - 雷达与转台中心尽量同轴
      - 固定牢固无晃动

      ------

      # Step 2：放置标定目标

      在指定距离处放置目标物，例如：

      ```
      5m
      10m
      20m
      ```

      推荐优先测试：

      ```
      10m以上
      ```

      因为距离越远，角度变化越明显。

      ------

      # Step 3：采集初始数据

      保持转台静止。

      雷达进行扫描，获取目标点云。

      记录：

      - 目标出现的连续点位
      - 起始 index
      - 结束 index
      - 中心 index
      - 对应距离 r1

      例如：

      ```
      目标点:
      1000
      1001
      1002

      中心index:
      1001
      ```

      ------

      # Step 4：控制转台旋转

      控制 Y100SC 转台旋转：

      ```
      0.1°
      ```

      等待转台稳定后再次采集数据。

      ------

      # Step 5：采集旋转后数据

      再次获取目标点云。

      记录：

      - 起始 index
      - 结束 index
      - 中心 index
      - 对应距离 r2

      例如：

      ```
      目标点:
      1001
      1002
      1003

      中心index:
      1002
      ```

      ------

      # Step 6：计算雷达角度变化

      计算点位变化：

      Δindex=index2−index1\Delta index = index_2-index_1Δindex=index2−index1

      计算雷达测得角度变化：

      Δθ=Δindex×0.1∘\Delta\theta = \Delta index \times 0.1^\circΔθ=Δindex×0.1∘

      例如：

      ```
      index1 = 1001
      index2 = 1002
      ```

      则：

      ```
      Δindex = 1
      Δθ = 0.1°
      ```

      ------

      # Step 7：计算角度误差

      理论值：

      ```
      0.1°
      ```

      实际值：

      ```
      Δθ
      ```

      计算误差：

      error=Δθ−0.1∘error = \Delta\theta - 0.1^\circerror=Δθ−0.1∘

      例如：

      ```
      实际:
      0.097°

      误差:
      -0.003°
      ```

# 五、建议增加重复测试

建议：

- 每个距离重复测试 10~20 次
- 统计：
  - 平均误差
  - 最大误差
  - 标准差

验证：

- 稳定性
- 重复性
- 抖动情况
